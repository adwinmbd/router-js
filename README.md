# ROUTER-JS

> Basic routing examples in vanilla JavaScript.

The project has been bundled with [Vite](https://github.com/vitejs/vite)

## Install:

```
npm install
```

## To run:

```
npm start
```

Opens in the browser on [http://localhost:3000](http://localhost:3000)

## To build:

```
npm run build
```

## To lint:

```
npm run lint
```

## To lint:

```
npm run format
```
